Todo List Web App

This is a simple Todo List web application built using HTML, CSS, and JavaScript. It allows users to add, manage, and delete daily tasks in an organized way.

Purpose

The main purpose of this project is to strengthen basic JavaScript knowledge through practical practice and improve understanding of real-world web development concepts.

Features
Add new tasks
Delete tasks
Store tasks using LocalStorage so data remains saved after refresh
Clean and responsive UI using Flexbox
Concepts Used
DOM Manipulation for creating and updating elements dynamically
Event Listeners to handle user actions like button clicks
forEach loop to display stored tasks efficiently
LocalStorage for saving and retrieving data in the browser
Flexbox for layout and responsive design
Program Flow
User enters a task in the input field
Button click triggers an event listener
JavaScript creates a new task element using DOM
Task is added to the list and stored in an array
Array is saved in LocalStorage for persistence
On page load, data is fetched from LocalStorage and displayed using forEach
